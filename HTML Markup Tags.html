HTML Markup Tags

Ryan Schubert

Web Page Section Tags

< !DOCTYPE html >
- Document type definition, goes at the top of file
< html > ... < / html >
- Start and end tag of an HTML document
< head > ... < / head >
- Start and end of the heading section
< body > ... < / body >
- Start and end of the body (content) of the document
< html lang = "en" >
- attribute in the html tag specifying that the document is written in English
< !-- comments -- >
- "< !--" starts a comment and "-- >" ends the comments, multiline allowed

Heading Section Tags

< title > ... < / title >
- Title of the document, displayed in the browser's title bar
< meta charset = "utf-8" / >
- Specifies the character set used in the document
- utf-8 is Unicode Transformation Format 8-bit ASCII compatible code

Body Section Tags

  1. Heading Tags
    • < h1 > ... < / h1 >
      • Heading level 1
    • < h2 > ... < / h2 >
      • Heading level 2
    • < h3 > ... < / h3 >
      • Heading level 3
    • < h4 > ... < / h4 >
      • Heading level 4
    • < h5 > ... < / h5 >
      • Heading level 5
    • < h6 > ... < / h6 >
      • Heading level 6
  2. Block Organization Tags
    • < p > ... < / p >
      • Paragraph
    • < br / >
      • Line break
    • < blockquote > ... < / blockquote >
      • Block Quotation
    • < div > ... < / div >
      • Creates a division (section) in a document
  3. List Tags
    • < ul > ... < / ul >
      • Unordered List
    • < ol > ... < / ol >
      • Ordered List
    • < li > ... < / li >
      • List item within an ordered or unordered list
    • < dl > ... < / dl >
      • Definition list
    • < dt > ... < / dt >
      • Definition term (item) within a definition list
    • < dd > ... < / dd >
      • Definition for a term within a definition list
  4. Link Tags
    • < a href = "url" > ... < / a >
      • Anchor link, link to another page
    • < a href = "url#name" > ... < / a >
      • Link to an anchor point, "name", within a page
    • < img src = "url" alt = "text" / >
      • Insert an image found at the URL, alternate text is hidden caption
    • < link rel = "text" href = "url" >
      • Creates a hyperlink
    • < script > ... < / script >
      • Defines a client-side script, such as a JavaScript
  5. Text Display Tags
    • < em > ... < / em >
      • Italics
    • < sub > ... < /sub >
      • Subscript
    • < sup > ... < /sup >
      • Superscript
    • < code > ... < / code >
      • Uses a fixed space, computer code font
  6. Table Tags
    • < table > ... < /table >
      • Table, contains table rows which contain table headings and data
    • < tr > ... < / tr >
      • Table row, contains headings and data
    • < th > ... < / th >
      • Table heading item
    • < td > ... < / td >
      • Table data item
    • < th colspan = "3" > ... < / th >
      • Table heading with column span of 3 columns (td also works)
    • < th rowspan = "4" > ... < / th >
      • Table heading with row span of 4 rows (td also works)
    • < caption > ... < / caption >
      • Gives a caption for the table
  7. Form Tags
    • < form action = "action" > ... < / form >
    • < input type = "text" name = "theName" size = "25" maxLength = "30 / >
    • < input type = "password" name = "myPassword" size = "10" maxLength = "10" / >
    • < input type = "checkbox" name = "buttonName" value = "dataValue" checked = "checked / >
    • < input type = "radio" name = "buttonName" value = " " checked = "checked" / >
    • < input type = "submit" value = "Submit Form" / >
    • < input type = "reset" value = "Reset Form" / >
